Installation
============

Install ```stitches``` using pip:

.. code-block:: console

  $ # stitches requires python >= 3.8.
  $ pip install stitches

You may also install ```stitches``` from source.

.. code-block:: console

  $ git clone https://github.com/JGCRI/stitches.git
  $ cd stitches
  $ pip install .
